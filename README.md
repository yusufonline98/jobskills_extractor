# JobSkills Extractor (LangChain) - Light Theme

Extracts skills from a Job Description and generates resume bullets.

## Features
- Categorizes skills: Technical, Tools, Soft, Certifications
- Generates 6 resume bullet points
- Simple HTML/CSS light-themed UI
- LangChain + LLM powered
